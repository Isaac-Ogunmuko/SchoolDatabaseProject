# School Database Project  

This is a Java school database project created for a programming course assignment. The program uses object-oriented programming concepts such as inheritance, constructors, arrays, ArrayLists, file input, and formatted output.

## Project Overview

The project models a simple school database system with different types of people and courses. It reads data from an input text file, creates objects, stores them in lists, and displays the database information in the console.

## Classes Included

* `Person.java` - Base class for people in the school database
* `Student.java` - Extends `Person` and represents a student
* `Employee.java` - Extends `Person` and represents an employee
* `Faculty.java` - Extends `Employee` and represents faculty members
* `GeneralStaff.java` - Extends `Employee` and represents general staff members
* `Course.java` - Represents a course
* `Driver_SchoolDB.java` - Main driver class that reads the file and displays the output
* `SchoolDB_Initial.txt` - Input file used by the program

## Concepts Used

* Java classes and objects
* Inheritance
* Constructors
* Getter methods
* Static variables
* ArrayLists
* File reading with `Scanner`
* Exception handling
* Formatted console output

## How to Run

1. Open the project in Eclipse, Visual Studio Code, or another Java IDE.
2. Make sure all `.java` files are inside the `schooldatabase` package.
3. Make sure `SchoolDB_Initial.txt` is included in the project.
4. Run `Driver_SchoolDB.java`.

## Example Output

The program displays sections for:

* Courses
* General Staff
* Faculty
* Students
